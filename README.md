# 用法

```
用法: SinglePython [OPTIONS]

Options:
-f <filename> --file=<filename>    输入 Python 文件名并运行 （*.py）
-h            --help               查看帮助
-v            --version            查看SinglePython版本
```

# 构建

1. 克隆此项目

```bash
git clone https://github.com/Han-Xiangming/SinglePython
cd SinglePython
```

2. 安装依赖

```bash
pip install -r .\requirement.txt
```

3. 构建 SinglePython
   <br/>运行build.bat即可
